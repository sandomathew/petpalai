# PetPalAI

PetPalAI is a Django-based AI assistant for pet owners, integrating LLMs for food analysis, pet management, and vet records.

## Stack
- Python 3.x, Django
- SQLite (for demo)
- Ollama LLM (`llama3.2`)
- HTML/CSS/JS
- Bootstrap
